## 공유 링크 토큰 설계

```
공유 링크 형식: /shared/{share_token}

share_token 구조 (JWT):
{
  "sub": "analysis_id",
  "type": "public" | "private",
  "team_id": "uuid",
  "exp": timestamp (7일 기본)
}

접근 제어:
  - public: 토큰 유효 시 비인증 사용자도 읽기 전용 접근
  - private: 같은 팀 멤버 + 유효 토큰 → 읽기/코멘트 가능
```

## 코멘트 API 명세

```
POST /api/test-cases/{id}/comments
Body: { "content": "텍스트 @user1 확인 필요", "mention_ids": ["uuid1"] }
Response: { "id": "uuid", "content": "...", "author": {...}, "mentions": [...], "created_at": "..." }

GET /api/test-cases/{id}/comments
Response: [{ "id": "uuid", "content": "...", "author": {...}, ... }]

DELETE /api/test-cases/{tc_id}/comments/{comment_id}
→ 작성자 본인 또는 QA 리더만 삭제 가능
```
